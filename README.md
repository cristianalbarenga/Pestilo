# Pestilo
Objeto que define y aplica estilo grafico 

- Pestilo.paleta( x );  
x = array de colores

- Pestilo.addEstilo( tipo , colorPrincipal , tamañoLinea );   
tipo = 'R': relleno | 'L': linea | 'C' combinado (relleno y linea)   
colorPrincipal = color de la forma   
tamañoLinea = grosor de linea

- Pestilo.limpiar();    
elimina el estilo
